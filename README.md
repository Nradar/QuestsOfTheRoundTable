# QuestsOfTheRoundTable
Carleton University COMP3004 term project, Quests Of The Roundtable
Team Members: Tiancheng Zhao(Leader), Xinting Wang, De Tao, Xin Wang

# Description
Multiplayer trading card game, allow four players to draw the the card, play the card in turn, and game rules are based on table trading card game: Quests of the Round Table.

# Features
AI involved in the game to substitue the player who is not attend, for example, 3 players can play with 1 AI player. AI player can draw cards and play the cards, select the senerio based on the current game situation.
